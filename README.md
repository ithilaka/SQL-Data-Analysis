# SQL Data Analysis

This repository contains a collection of SQL data analysis projects. Each project focuses on using SQL to explore data, answer business questions, calculate key metrics, and generate insights.

## Projects

### 1. E-commerce Data Analysis

This project analyzes e-commerce business data using SQL. It covers customer behavior, sales performance, discount impact, product and inventory trends, store performance, staff performance, RFM customer segmentation, and year-over-year growth analysis.

[View Project](./E-commerce)

## Skills Demonstrated

- SQL querying
- Data cleaning and exploration
- Joins and relationships
- Aggregations
- Window functions
- KPI analysis
- RFM analysis
- Year-over-year performance analysis
- Query optimization
- Business insight generation
